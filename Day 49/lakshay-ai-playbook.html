What this app does

My AI Playbook is your personal system for reusing what already works with Claude, instead of rewriting prompts from scratch every time. Workflows are ready-to-fill templates for your recurring tasks (like turning dashboard numbers into a stakeholder-ready summary). The Prompt Builder lets you assemble a custom prompt from labeled building blocks, with a live preview. The Loop Builder turns any prompt into a self-checking, iterative one. Everything you save stays in this browser — nothing is sent anywhere.

    What is My AI Playbook?

    A quick tour of every section
    Dashboard

    A landing page showing your workflow counts and quick links into the tool.

    Workflows

    Reusable prompt templates for your recurring tasks. Each one has fillable variables (dropdowns where possible), a live preview, a worked example, and best practices — so you fill in a few fields instead of rewriting a prompt from memory.

    Prompt Builder

    For a one-off prompt that doesn't fit a saved workflow. You stack labeled blocks — Role, Objective, Context, Constraints, Reasoning strategy, Output format, Tone, Examples, Quality checks — and watch the full prompt build itself in the preview pane. You can save the result as a new workflow.

    Loop Builder

    Converts a normal prompt into a self-checking one: Claude produces an answer, scores it against criteria you define, revises, and stops when it passes or hits a limit you set — so it doesn't loop forever or drift off-task.

    Favorites & Import/Export

    Star anything for quick access. Everything lives in this browser's local storage — export it as a backup or to move to another machine.
